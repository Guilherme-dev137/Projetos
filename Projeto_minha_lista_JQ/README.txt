Projeto desenvolvido com finalidade de praticar os conchecimentos adiquiridos com o estudo da técnologia JQUERY
Autor: Guilherme Gonçalves

Funcionalidade:
Essa aplicação a principal função dela é a criação de uma lista
Existem três botões seque a explicação de cada um:
	Registrar: O botão "registrar" irá adicionar o item digitado na lista, e criará toda a interface da lista
	Limpa: Cada elemento adicionado na lista pode ser apagado individualmente, para isso basta selecionar o item a ser
	deletado e clicar no botão "limpa", isso irá excluir todos os elementos adicionados
	Limpar Lista: Esse botão limpará toda a lista, independente de estar com os item selecionados ou não

Observação:  Como o código foi feito para teste do curso, não foi desenvolvido a responsividade da página, mas logo o projeto estará responsivo para todos os aparelhos
