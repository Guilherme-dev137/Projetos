Projeto sorteio de rifa
Esse projeto tem como objetivo realizar um sorteio de uma pessoa que tenha sido cadastrada no sistema

Dicas de utilização do sistema:

Cadastro de pessoas:
	Para realizar o sorteio das pessoas basta utilizar a barra principal para registrar as pessoas, para simplificar e tornar o cadastro mais rápido, é possível realizar o cadastramento dessa forma: "pessoa1 pessoa2 pessoa3", fazendo dessa forma é possível cadastrar três pessoas de uma vez, o sistema faz a separação do nome por meio do espaço, caso tenha alguma pessoa com dois nomes ou com um sobrenome, basta seguir esse modelo: "nome.sobrenome".

Realização do sorteio:
	Para realizar o sorteio só será possível o mesmo ser realizado com duas ou mais pessoas cadastradas no sistema.

Observações:
	Essa página realiza o sorteio apenas por nome, onde o nome de várias pessoas é cadastrada e apenas uma é sorteada, em breve será desenvolvida uma nova plataforma com mais opções de sorteios.
